# 📝 TaskFlow - AI Assisted Task Management App

TaskFlow is a modern Task Management application built with **React**, **Vite**, and **TypeScript**. The project was developed with AI assistance while following a structured development and review process.

## 🚀 Features

- ✅ Create tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 📋 Task status (To Do, In Progress, Done)
- 🔍 Search tasks
- 🎯 Filter tasks by status
- ⚡ Priority levels (Low, Medium, High)
- 📅 Due date support
- 🏷️ Task tags
- 🌙 Dark / ☀️ Light theme
- 💾 Local Storage persistence
- 📱 Responsive design

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS

## ▶️ Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
