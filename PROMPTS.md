# AI Prompts Used

## Prompt 1 (Initial)

Create a simple Task Management application using React and TypeScript where users can add and delete tasks.

---

## Prompt 2 (Improved)

Create a professional Task Management application using React, Vite and TypeScript.

Requirements:

- Use reusable React components.
- Use TypeScript interfaces.
- Users can add, edit and delete tasks.
- Add task status (To Do, In Progress, Done).
- Include task priority.
- Include due dates.
- Add search functionality.
- Add filters for task status.
- Save tasks using localStorage.
- Create a responsive modern UI.
- Add Dark/Light mode.
- Use clean CSS without external UI libraries.
- Review the generated code and identify possible improvements before finalizing.

---

## Additional Prompts

- Improve accessibility of the application.
- Refactor repetitive code into reusable components.
- Improve responsiveness for smaller screens.
- Suggest better component organization.
- Improve form validation.
